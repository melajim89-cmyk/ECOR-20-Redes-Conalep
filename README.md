# 🌐 ECOR-20 - Manejo de procedimientos y comunicación en redes

Repositorio académico del módulo ECOR-20 del CONALEP NAUCALPAN I.

## 📚 Descripción
Este repositorio contiene prácticas, configuraciones, simulaciones y evidencias del módulo:

**Manejo de procedimientos y comunicación en redes**

## 🎯 Propósito
Desarrollar habilidades en:
- Configuración de switches y routers
- Direccionamiento IPv4 e IPv6
- Diseño de redes
- Uso de protocolos de comunicación

## 🧠 Contenido

### 🔹 Unidad 1: Configuración inicial de dispositivos
- Configuración de switches
- Capa física y enlace de datos
### 🔹 Unidad 2: Direccionamiento IP
- Ethernet
- IPv4 / IPv6
- Configuración de routers

### 🔹 Unidad 3: Diseño de red
- Protocolos de transporte
- Seguridad en redes
- Implementación de red

## 🛠 Tecnologías usadas
- Cisco Packet Tracer
- Excel (subneteo)
- Markdown
- Simulaciones de red
